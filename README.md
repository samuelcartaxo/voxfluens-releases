##  How to Download and Install

Getting started is effortless. Download the latest installer from our releases directory and install in minutes.

### Step 1: Download the Installer
- Visit the [Releases Page](https://github.com/samuelcartaxo/voxfluens-releases/releases).
- Download the latest **VoxFluens-Setup-{version}.exe** from the `setup/` directory (e.g., `VoxFluens-Setup-2.0.1.exe`).
- This is the full installer  no manual ZIP extraction needed.

### Step 2: Run the Installer
- Double-click the downloaded `.exe` file.
- Follow the on-screen prompts:
  - Accept the End User License Agreement (EULA).
  - Choose installation directory (default: `C:\Program Files\VoxFluens`).
  - Select components (main app, shortcuts, locale files).
- The installer creates desktop shortcuts and integrates with Windows seamlessly.

### Step 3: Launch and Configure
- Open VoxFluens from the desktop shortcut or Start Menu.
- On first run, choose Guest Mode to try for free, or sign up for full access.
- Configure hotkeys, audio devices, and language in the dashboard.

**That\'s it! You\'re ready to transcribe. If you encounter issues, check our [troubleshooting guide](https://github.com/samuelcartaxo/vf_app2/wiki/Troubleshooting).**
